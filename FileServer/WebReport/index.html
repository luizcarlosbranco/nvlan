Dúvidas sobre armazenamento de dados


Quais são os tipos de dados?


Os tipos de dados são referentes à sua utilidade:

• Individuais: Que não é utilizado pela empresa, nem necessário para as tarefas.
• De equipes e projetos: Para trabalhos entre as equipes e nos projetos das áreas e da empresa.
• Da empresa: Dados utilizados pela e/ou para a empresa.

O que devo guardar no servidor de arquivos?


Você deve guardas os dados que são utilizados pela e/ou para a empresa, mas não deve colocar dados individuais (mesmo que parecam utéis) no servidor enquanto o mesmo não tiver uma utilidade.

Qual a diferença entre um dado individual e um dado da empresa?


A diferença é a seguinte: Se o dado é utilizado pela e/ou para a empresa, mesmo que ele só seja utilizado por uma pessoa (como um dado que um colaborador precisa para executar suas atividades) ele é um dado da empresa.

Por outro lado, se o dado nao é utilizado pela e/ou para a empresa, mesmo que seja um dado interessante ele é apenas um dado individual.

Qual a diferença entre guardar os arquivos no computador ou no servidor?


Os arquivos armazenados no servidor são utilizados nas atividades de sua empresa, sendo assim:

• Eles são compartilhados para aqueles que usam para relizar as atividades
• São feitos backups
• A equipe de TI atua para manter esses dados disponiveis, integres e seguros.

Posso armazenar dados coporativos individuais no servidor de arquivos?


A resposta é não. Esses dados não devem ser amazenados no servidor de arquivos da empresa. Eles devem ficar armazenados:

• No OneDrive do colaborador
• Em uma unidade de armazenamento externa do setor (caso o setor opter em adquirir)
• Na pasta TEMP da rede (que possuí uma rodina de limpeza semanal automática)

Porque não apenas armazenar tudo no servidor de arquivos?


Ao colocar dados dispensáveis e que não é utilizado pela e/ou para a empresa, você está consumindo espaço que outras áreas poderiam utilizar. Além disso, quando você armazena esse tipo de arquivos no servidor, você está atribuindo tarefas à coordenação de informática e retirando um recurso comum a todas as outras areas de sua empresa, por isso nunca faça isso.

ATENÇÃO: NÃO É PORQUE O DADO É INTERESSANTE E ESTÁ DENTRO DO AMBIENTE COMPUTACIONAL DA SUA EMPRESA QUE VOCÊ DEVE CONSIDERAR QUE ESSE DADO DEVE FICAR NO SERVIDOR DE ARQUIVOS, ANALISE O TIPO DE DADO DE ACORDO COM OS TRES TIPOS DEFINIDOS ACIMA.
